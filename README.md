## Contribute to this list!

欢迎提交新的 awesome-ipfs 项目。 为了向这个列表添加一个元素，您需要修改`/data` 运行一个简单的 `make build` 在发布您的 pull 请求之前，因为 Readme 和站点都是自动生成的， 参考相关的[CONTRIBUTING.md guidelines](https://github.com/ipfs/awesome-ipfs/blob/master/CONTRIBUTING.md)指导方针

**注意**: 如果您想直接编辑 README 文件(而不是项目 / 链接) ，您需要修改 [README template](https://github.com/ipfs/awesome-ipfs/blob/master/scripts/readme-template.md)模板，然后运行 `make build`
